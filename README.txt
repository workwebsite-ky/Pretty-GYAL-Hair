PRETTY GYAL HAIR LLC — WEBSITE
==============================

HOW TO VIEW
-----------
Unzip and double-click index.html — it opens in any browser.
To publish: upload the whole folder to any host (Netlify, Vercel, GoDaddy,
Hostinger, etc.). No build step needed — pure HTML/CSS/JS.

PAGES
-----
index.html           Home (hero, featured hair, why us, gallery, reviews, FAQ)
shop.html            Shop Hair — full price lists (bundles / frontal wigs / bob wigs)
custom-wigs.html     Custom Wigs — process, custom color gallery
about.html           About Us — brand story, mission/vision, CEO
cares.html           Pretty GYAL Cares — community initiative
hair-show.html       Hair Show — 2nd Annual Showcase w/ live countdown
collaborations.html  Collaborations — partner pitch form
contact.html         Contact / Book Now — form, map, details

EASY EDITS
----------
- Colors: css/style.css, top ":root" block (--orange is the logo color #E95924)
- Prices: shop.html — plain tables, edit any number
- Hair show date/countdown: hair-show.html — search "data-countdown"
- Phone/email/socials: appear in footer of every page — find & replace:
    +1 917-485-1232 / prettygyalhair@gmail.com
- Contact forms open the visitor's email app addressed to prettygyalhair@gmail.com
  (mailto). For a hosted form, swap in Formspree/Getform later.

IMAGES
------
All images are in /images (client photos from IG/FB + logo + flyers).
Replace any image with a new file of the same name to update the site.
